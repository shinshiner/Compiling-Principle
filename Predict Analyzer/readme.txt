本程序实现了从文本读入上下文无关文法（可含直接左递归），构造其预测分析表并输出到文本文件中
本程序所使用的方法与书上描述相同，先构造FIRST和FOLLOW集合，再据此构造预测分析表，其中FIRST集合的构造我并没有显式地构造终结符的FIRST集合，只是在用到的时候对符号类型做出判断，再相应取FIRST集合值或其本身，其余与书上思想完全相同。
输入格式：第一行为一个整数，表示产生式数量，接下来n行是对应的n个产生式，格式为：左部符号+空格+"->"+空格+右部，右部的符号用空格隔开，若有多个推导则用"|"隔开。
考虑到邮件大小，仅发送cpp文件，若cpp文件无法编译运行，可在VS里新建项目并复制代码后编译。
输入输出样例：
输入：
5
E -> T E'
E' -> + T E' | ~
T -> F T'
T' -> * F T' | ~
F -> ( E ) | id
输出（最大化文件看效果更佳）：
-------------------------------------------------------------------------------------
|          |$          |(          |)          |*          |+          |id         |
-------------------------------------------------------------------------------------
E          |           |E->T E'    |           |           |           |E->T E'    |
-------------------------------------------------------------------------------------
E'         |E'->~      |           |E'->~      |           |E'->+ T E' |           |
-------------------------------------------------------------------------------------
T          |           |T->F T'    |           |           |           |T->F T'    |
-------------------------------------------------------------------------------------
T'         |T'->~      |           |T'->~      |T'->* F T' |T'->~      |           |
-------------------------------------------------------------------------------------
F          |           |F->( E )   |           |           |           |F->id      |
-------------------------------------------------------------------------------------

输入（直接左递归）：
2
A -> A a | c
B -> b
输出：
-------------------------------------------------------------
|          |$          |a          |b          |c          |
-------------------------------------------------------------
A          |           |           |           |A->c A'    |
-------------------------------------------------------------
B          |           |           |B->b       |           |
-------------------------------------------------------------
A'         |A'->~      |A'->a A'   |           |           |
-------------------------------------------------------------

消除左递归结果：
A->c A'
B->b
A'->a A'|~

若输入有误（假定格式正确，错误出现于逻辑上，如无限推导），则程序不可避免地会输出错误结果。
输入：
2
A -> A abc | A id | A'
E -> E
输出：
-------------------------------------------------
|          |$          |A'         |abc        |id         |
-------------------------------------------------
A          |           |           |           |           |
-------------------------------------------------
E          |           |           |           |           |
-------------------------------------------------
A''        |           |           |           |           |
-------------------------------------------------
E'         |           |           |           |           |
-------------------------------------------------
E'         |           |           |           |           |
-------------------------------------------------

输入（课本上的反例）：
3
S -> i E t S S' | a
S' -> e S | ~
E -> b
输出：
-------------------------------------------------------------------------------------
|          |$          |a          |b          |e          |i          |t          |
-------------------------------------------------------------------------------------
S          |           |S->a       |           |           |S->i E t S S'|           |
-------------------------------------------------------------------------------------
S'         |S'->~      |           |           |S'->~      |           |           |
-------------------------------------------------------------------------------------
E          |           |           |E->b       |           |           |           |
-------------------------------------------------------------------------------------
